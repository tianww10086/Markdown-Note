## 3.1 枚举

​	枚举——又称暴力枚举，面对有些问题是最简单粗暴的解法：**枚举出所有可能的情况，然后判断或者统计，从而解决问题。**

​	



#### 3.1.1 寻找质数

代码：

```java
import java.util.Scanner;


public class Main {
    private static final Scanner in = new Scanner(System.in);

    public static void main(String[] args) {
        int count =0;
        boolean flag = true;
        //求一个数是不是质数,质数只能被1和本身整除，
        int n = 0;
        while(true){
            if(isPrime(n)){
                count++;
                if(count==2025){
                    System.out.println(n);
                    break;
                }
            }
            n++;
        }

    }

    public static boolean isPrime(int n){
        if(n<=1)
            return false;
        if( n==2) //2是唯一的偶质数
            return true;

        //排除其他偶数
        if(n%2==0)
            return false;

        //只需检查奇数因子，范围到sqrt(n)
        int lim = (int)Math.sqrt(n);
        for(int i=3;i<=lim;i+=2)
        {
            if(n%i==0)
                return false;
        }

        return true;
    }
}

```

​	我们关注一下这个求质数的方法。这是一个优化过后的方法。它能接受所有数，从而判断它是不是质数。请注意到，`for`循环的范围是从3（3以前的数在前面被处理了）`lim`（包括本身）。这是为什么，我们先给出普通解法：

```java
    public static boolean isPrime(int n){
        if(n<=1)
            return false;
        if( n==2) //2是唯一的偶质数
            return true;

        //排除其他偶数
        if(n%2==0)
            return false;

        //只需检查奇数因子，范围到sqrt(n)
        for(int i=3;i<lim;i++)
        {
            if(n%i==0)
                return false;
        }

        return true;
    }
```

​	普通解法暴力枚举了每一个可能，这种写法适合比赛得分，但在时间复杂度比较高，属于`O(n)`的时间复杂度。有没有办法优化它，当然有，上面的解法就是优化后的解法。重新回答优化版本。**为什么范围是3~sqrt(n)**。

​	下面总结一个规律：

​	在判断一个数`n`是否为质数时，我们只需要从2到$\sqrt{n}$（闭区间）之间是否存在能整除n的数，这是因为：

- 如果n是合数，那么它一定可以写成两个大于1的正数相乘的形式： $n = a * b$。

- 假设 $a<= b$，则a必然小于或等于$\sqrt n$ 。因为 如果$a> \sqrt n$ ，那么 $a*b >  \sqrt n *  \sqrt n = n$。这与等式矛盾

- **因此，如果n存在因数，那么至少有一个因数$<=  \sqrt n$。我们只需要检查这个范围内是否有因数即可**。如果从2到$ \sqrt n$都没有找到因数，那么n就是质数。

  这种方法的正确性在于：**只要有一个因数大于 $\sqrt n$，它对应的另一个因数必然小于 $\sqrt n$，所以我们不会漏掉任何因数**。

---

*合数：除了1和它本身之外，还能被其他正整数整除的自然数*

---

​	举例：对于一个合数，它一定可以写成a\*b的形式。设a一定<=b。则a必然小于或等于 $\sqrt n$。比如数12

- 1*12 = 12

- 2*6 = 12

- 3*4 = 12

-  $\sqrt 12* \sqrt 12 = 12$

  我们让a始终小于或等于b，列出了三种情况。符合如果n存在因数，那么至少有一个因数$<=  \sqrt n$。很明显这个数有多个因数。同样也符合只要有一个因数大于 $\sqrt n$，另外一个因数必然小于 $\sqrt n$。这里 $\sqrt 12$ ~=3.46....。所以只需要遍历到这个数的平方根即可。如果能够被整除，就代表它不是质数

​	

​	举例2：现在把这个数举例成一个质数。如7。那么它的平方根是：2.64....。很明显，它不存在处除1和本身其他因数。这是一个技巧，记住就行了。
