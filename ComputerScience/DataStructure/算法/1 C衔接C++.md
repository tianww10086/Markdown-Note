## 1.1 C与C++的差异

​	C++是C语言的超集，它包含了C语言里的内容，也有C语言里没有的东西，所以，这一章节主要讲述C语言里没有的，而C++有的。由于C++有STL(标准模板库），里面包括很多容器、算法和函数。在参见算法比赛时很有优势，遇见相关的题目可以直接调用容器。

​	在讲述STL之前，必须先讲清楚c++和c的基本内容有哪些不同。





#### 1.1.1 数据类型

​	C语言中，没有bool类型，就算有bool，也是用宏定义来表示，用1表示true，用0表示false，这点和C++一样。而C++里包括了大部分主流的类型。C语言里用`char`数组表示字符串。并用`\0`表示字符串结束。当然C++里也可以这么写，但C++有专属的字符串类。后面会介绍字符串、数组、指针、结构体等等在C++里怎么使用。



#### 1.1.2 简单输入输出

​	下面介绍C++程序的基本结构

```c++	
#include<iostream>  // 导入标准输入输出流头文件

using namespace std;  //使用标准名称空间
int main(){ //主函数，与C语言无异
    cout<<"HelloWorld"<<endl; //输出并换行
}
```

`using namespace std`表示使用`std`名称空间，是为了避免开发多重项目，名字重复的问题，大多数标准名称都在这个空间里。

​	输出函数：

```c++	
cout<<"字符串"<<endl;
```

​	输入函数:

```c++
int a;
cin>>a;
```

> ​	注意，cin输入会跳过空格、制表符和换行符等字符。

​	当你想要读取这些字符时，请使用`cin.get(char)`函数，该函数逐个字符读取输入的每个字符。包括**空格、制表符、换行符**。使用方式如下：
```c++
char ch;
cin.get(ch);
```





#### 1.1.3 for的变式

​	C++11新增了一种循环：基于范围的for循环。这种循环能够快速的访问数组（或容器类）。它的基本格式如下：

```c++
for(元素类型 变量名：容器/数组){
    //使用变量名
}
```

​	如下

```c++
int arr[]{1,9,8,7,3,6};
for(auto a:arr){
    cout<<a<<" ";
}
```

​	最初，a代表数组的第一个元素，然后不断执行循环，a依次表示数组的其他元素。需要注意的是，如果要修改数组的元素，则需像下面一样使用：

```c++
for(auto &a :arr){
    //.....
}
```

​	符号&表明x是一个引用变量。

​	还可以结合使用初始化列表：

```c++
for(int x:{3,5,9,8,6})
    cout<<x<<endl;
```



## 1.2 复合类型

​	复合类型，是基于基本类型创建的类型，如数组、字符串、结构体、指针等。



#### 1.2.3 数组

​	数组的静态创建

```c++
int arr[10]; 
//初始化方法1
int arr[10]={1,5,7,9,....};
//初始化方法2
int arr[10]{1,3,4,5,...};
//不指定数组大小初始化
int arr[]={1,4,5,7,9};
```

​	数组的动态创建：

```c++
//一维数组
int *arr = new int[10];

// 第一步：分配行指针数组 - 创建10个int*指针，每个指针将指向一行
int **twoArr = new int*[10]; 

// 第二步：为每一行分配列空间
for(int i = 0; i < 10; i++) {
    // 为第i行分配10个整数的连续内存空间
    twoArr[i] = new int[10];
    
    // 此时 twoArr[i] 指向一个包含10个int的数组
    // 可以通过 twoArr[i][j] 访问第i行第j列的元素
}
```

​	用一个指针指向分配的内存。**`new`运算符返回第一个元素的地址**。使用完`new`运算符分配的内存块，应使用`delete`删除它们。对于数组内存块，使用下面的格式释放：

```c++
delete []arr;
// 释放二维数组,先释放内层
for(int i=0;i<n;i++){
    delete[] arr[i];
}
//再释放外层
delete[] twoArr;
```

​	注意，new分配的内存是在堆里面。
