## 2.1 模拟

​	所谓模拟，是指让程序完整地按照题目叙述的方式运行得到最终的答案。



2.2.1 兵乓球

```java
import java.io.IOException;
import java.util.Scanner;

public class Main {
    static int[] f = {11,21}; //两种赛制
    static int[] a = new int[25*2500+10];
    static int n=0;
    public static void main(String[] args) {

        Scanner in = new Scanner(System.in);

        int temp;
        try{
            while((temp = System.in.read()) !=-1){
                char ch =(char)temp;
                if(ch=='E')
                    break;
                if(ch=='W')
                    a[n++]=1;
                if(ch=='L')
                    a[n++]=0;
            }
        }catch(IOException e){
            e.printStackTrace();
        }



        for(int k=0;k<2;k++){
            int w=0, l=0; //记分
            for(int i=0;i<n;i++){
                w+=a[i];
                l+=1-a[i];//当a[i]值是0时更新1，反之不更新
                // 获胜者超过对应分数且超出对手两分
                if( Math.max(w,l)>= f[k] && Math.abs(w-l)>=2 ){
                    System.out.println(w+":"+l);
                    w=l=0; //重置
                }
            }

            //未完成的比赛也要输出结果
            System.out.println(w+":"+l);
            System.out.println();
        }
    }
}
```
