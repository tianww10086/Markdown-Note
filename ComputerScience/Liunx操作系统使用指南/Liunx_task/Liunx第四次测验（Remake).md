## 2.usermod

​	使用该命令修改用户名并修改所属用户组。

### 2.1 修改用户名

![image-20251120085216597](../IMG/image-20251120085216597.png)

​	将u02用户名修改为u03，可选项使用`-l`



### 2.2 修改所属用户组

​	将u03的用户修改到u01用户组去。可选性使用`-g`

![image-20251120085530839](../IMG/image-20251120085530839.png)



### 2.3 锁定账户

​	锁定后该账户暂时禁止登录系统。可选项`-L`。该用户在`/etc/shadow`密码配置文件中密码字段会出现“!”

![image-20251120090035452](../IMG/image-20251120090035452.png)



### 2.4 解除锁定

​	可选项`-U`，相反的，密码字段钱的`!`会消失。锁定账户后，若账户没设密码，会需要先设置密码再解除锁定

![image-20251120090515550](../IMG/image-20251120090515550.png)

​	先使用`passwd`命令设置密码。

​	<img src="../IMG/image-20251120091432938.png" alt="image-20251120091432938" style="zoom:50%;" />

​	这里设置`242656LOVE`

​	在使用`usermod -U u03`解除锁定

<img src="../IMG/image-20251120091607009.png" alt="image-20251120091607009" style="zoom:50%;" />

​	可以看见`u03`密码字段前面的“！”消失了。`u01密码为:24265601`

## 5. group

​	每个用户都有一个用户组。一个用户组可以包含多个用户，可以对它进行增删改查。



### 5.1 增加用户组(groupadd)

​	现在增加一个`normal`用户组，可选项`-r`。表示创建系统用户组，其组ID小于500；普通用户组ID大于或等于500

![image-20251120093224253](../IMG/image-20251120093224253.png)

​	可以看见该用户组id为：`1005`





### 5.2 修改用户组名(groupmod)

​	可选参数`-n`。第一个名字表示新名字，第二个名字表示原用户组名。将`normal`用户组修改为`common`用户组

![image-20251120093351740](../IMG/image-20251120093351740.png)

​	只是修改了用户组名。用户组id并没有修改。





### 5.3 设置用户组ID

​	可选项`-g`,用法：`groupmod -g NewGid groupname`

```bash
groupmod -g 1101 common
```

![image-20251120093607207](../IMG/image-20251120093607207.png)



### 5.4 为用户组添加用户

​	可选项`-a`，用法：`gpasswd -a 用户名 用户组名`

```bash
gpasswd -a commonUser01 common
```

![image-20251120094111916](../IMG/image-20251120094111916.png)



### 5.5 从用户组删除用户

​	可选项`-d`。用法`gpasswd -d 用户名 用户名`

```bash
gpasswd -d commonUser01 common;
```

![image-20251120095001706](../IMG/image-20251120095001706.png)



### 5.6 设置用户组管理员

​	可选项`-A`，用法`gpasswd -A 用户名 用户组名`

```bash
gpasswd -A commonUser01 common;
```

![image-20251120095706755](../IMG/image-20251120095706755.png)



### 5.7 删除用户组

​	用法：`groupdel 用户组名`

```bash
groupdel common;
```

![image-20251120095958422](../IMG/image-20251120095958422.png)
