## 4.1 vue-cli

​	Vue-cli是vue官方提供的脚手架，用于快速生成一个Vue项目模板。提供如下功能：

1. 统一的目录结构
2. 本地调试
3. 热部署
4. 单元测试
5. 集成打包上线。

​	使用`npm`工具输入如下命令安装脚手架：

```HTML
npm install -g @vue/cli
```

​	

### 4.1.1 部署Vue项目

​	Vue提供两种部署标准vue项目的方式，一是ui界面。二是命令行。现在介绍命令行部署方式

```bash
vue create my-project #创建一个vue2项目
```

​	部署完成后，可以用命令`npm run serve`运行该项目

```bash
npm run serve
```



## 4.2 Element

​	Element是饿了吗开发的一个外观组件库。安装教程如下。在项目所在地址终端运行：

```bash
npm install element
```

​	element对应vue2，如果是vue3的项目则需安装`element-plus`

​	完整导入，在`main.js`文件中：

```bash
import { createApp } from 'vue'
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'
import App from './App.vue'

const app = createApp(App)

app.use(ElementPlus)
app.mount('#app')
```



### 4.2.1 eTable

​	表格组件模板：

```vue
  <el-table :data="tableData" height="250" stripe style="width: 100%" border =ture>
    <el-table-column prop="date" label="日期" width="180" />
    <el-table-column prop="name" label="名字" width="180" />
    <el-table-column prop="address" label="地址" />
  </el-table>
```

​	`border`属性为true时有竖线边框。



#### 4.2.2 Pagination

