# 1. HTML基本结构

​	HTML是超文本标记语言，主要由标签组成

```HTML
<html>
    <head>
        <title>文本标题</title>
    </head>
    <body>
        //正文
    </body>
</html>
```

​	所有HTML文件都由<html></html>包含，用于标记这个文件是HTML文件<head></head>说明文件的标题和整个文件的公共属性。<body></body> 包含网页主体结构。



### 1.1网页的注释

​	网页 注释：`<! -- 注释的文字 -->`

​	CSS注释：`/**/`

​	JavaScript注释：`//`





### 1.2 标题标签

​	`<h1></h1>`表示一级标题，`<h2></h2>`表示二级标题，以此类推。最小的是六级标题`<h6></h6>`

```html
<h1>
    这是一个一级标题
</h1>
```



### 1.3段落标签

​	`<p></p>`标签用于将文字段落化，每一个段落标签代表一个段落。段落与段落之间自动换行。段落没有首行缩进
